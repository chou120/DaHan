```text
被static修饰会被加载到内存中的方法区中去
    如果是静态代码块  那就是在加载的时候就已经执行了
    如果是 静态方法或者是静态的成员变量 那就是被调用的时候才会被执行或者显示
```

```text
内部类的分类:
1.成员内部类
      可以访问外部类的所有的属性和方法,静态方法直接访问
2.静态内部类
       在成员内部类多加一个  static 关键字
       静态内部类只能访问外部类的静态成员

3.匿名内部类
       如果只对抽象类里面的方法或者接口里面的方法只使用一次没有其他地方的调用 那就没必要重新写一个class
  (子类)

4.局部内部类
      定义在方法内部的 作用范围只在方法里面有效


内部类有什么作用:
    1.可以很好的实现隐藏
    2.间接的实现多继承
```
